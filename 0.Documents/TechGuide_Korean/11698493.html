Technical Documents(한국어) : Altibase APRE(SES) *C/C++ Makefile

Created by Shin beum-su, last modified by heejung lee on Dec 12, 2019

개요

 본 문서는   Unix 시스템에서 Altibase 가 제공하는 APRE(구SESC)*C/C++ Precompiler를 이용하여 컴파일하는 방법과 컴파일 단계별로 발생할 수 있는 문제들에 대한 해결 방법을 설명한다.

본 문서에 사용된 예제와 설명은 사용자층이  넓은  OS인 Linux 와 gcc 컴파일  환경에서 작성되었으며  다른 UNIX 환경(HP, SUN, AIX) 의   기타 컴파일러 사용자를 위해서도  참고할 수 있도록 필요한 정보를 추가 기술하였다.

 아래 문서들을 사전에 읽어 볼 것을 권장한다.

  1. Altibase Precompiler 가이드
  2. Altibase APRE New Features 업그레이드 가이드

본 문서의 테스트 환경은 다음과 같다.

  • ALTIBASE : Altibase 6 이상 버전
  • OS : Linux ( 2.6.32-504.el6.x86_64 )
  • 컴파일러 버전 : gcc version 4.4.7 20120313 (Red Hat 4.4.7-11) (GCC)
