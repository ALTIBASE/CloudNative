Technical Documents(한국어) : Altibase를 위한 SQuirrel SQL Client Quick 가이드

Created by Shin beum-su, last modified by heejung lee on Dec 12, 2019

개요

본 문서에서는  SQuirrel SQL Client 를  설치하고   Altibase  JDBC를 통해서  Altibase와 연동하는 방법을 설명한다.

 

본 문서의 테스트 환경은 다음과 같다.

  • ALTIBASE : Altibase 6 이상 버전
  • OS : Linux ( 2.6.32-504.el6.x86_64 )
  • SQuirrel : 3.7.1 standard
