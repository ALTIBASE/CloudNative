FAQ(한국어) : Stored Procedure 내에서 DML 로 영향 받은 레코드 수 확인 방법

Created by heejung lee on Jan 21, 2015


SQL%ROWCOUNT 를 사용하면 DML 로 영향을 받은 레코드 수를 알 수 있습니다.


수행 예
CREATE OR REPLACE PROCEDURE proc1
AS
  v1 INTEGER;
BEGIN
  DELETE FROM MEM_T LIMIT 10 ;
  v1 := SQL%ROWCOUNT;
  PRINTLN(v1);
END;
/

수행 예) 

iSQL> exec proc1;
10
Execute success.
