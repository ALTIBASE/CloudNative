Technical Documents(English) : 2. Altibase JDBC Driver Registeration

Created by richard.nahm on Aug 25, 2020

2.1 How to register

  1. Double-click "squirrel-sql.jar" in the directory where SQuirrel SQL Client is installed to run the program.
  2. Click Drivers -> New Driver to register Altibase JDBC driver. If the Drivers menu is inactive, activate it by clicking Windows -> View Drivers.

  3. Enter the following in the Driver item of the Add Driver window.

    Name: Altibase

Example URL:jdbc:Altibase://<host>:<port>/<database>
  4. Click the "Add" button in the Extra Class Path tab at the bottom and select the Altibase JDBC drive file to use.
  5. After selecting the file, click the "List Drivers" button, and check if "Altibase.jdbc.driver.AltibaseDriver" is added to the Class Name combo box at the bottom.
  6. Click the "OK" button to complete the Altibase JDBC driver registration.
