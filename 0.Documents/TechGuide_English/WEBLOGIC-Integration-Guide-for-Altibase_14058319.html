Technical Documents(English) : WEBLOGIC Integration Guide for Altibase

Created by richard.nahm on May 15, 2020

Overview

This is a technical document for integrating with Altibase and WebLogic. It also covers running a simple web application that works with Altibase on the assumption that WebLogic is installed.

This document is based on WebLogic 12.1.3.0 on the Windows platform, and all contents may differ depending on the version. For basic environments such as WebLogic installation and domain creation, refer to Oracle's documentation and management console help provided when installing WebLogic (Administration Console).
