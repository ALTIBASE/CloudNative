Technical Documents(English) : 1. SQuirrel SQL Client Installation

Created by richard.nahm on Aug 25, 2020

This chapter describes how to install SQuirrel SQL Client. This document was prepared by downloading "Install jar of SQuirrel 3.7.1 for Windows/Linux/others" in the Windows environment.

1.1 Installation procedure

  1. Download the SQuirrel SQL Client suitable for the user from the link below.
    http://squirrel-sql.sourceforge.net/#installation
  2. Double-click the downloaded "squirrel-sql-3.7.1-standard.jar" to start the installation
    https://jaist.dl.sourceforge.net/project/squirrel-sql/1-stable/3.7.1/squirrel-sql-3.7.1-standard.jar
