WebLogic Version

We recommend to use the latest version, and at least WebLogic 8.1.6.0 or later is recommended.

If a different version of Altibase is integrated at a lower version than this, the JDBC driver integration may not work properly, which may cause the operation of the WebLogic server instance to fail or hang.

JDBC Specification

The Altibase 5 JDBC driver was built in JDK 1.4 based on the JDBC specification 3.0 (JRE 1.4 or 1.5), and returns "Not supported" or an error in case of non-spec support.

Therefore, an application program written on the basis of JDBC specification 4.0 (JRE 1.6 or later) may cause problems due to unsupported specification, so it is necessary to comply with JDBC 3.0 specification from the development stage.
