FAQ(English) : ERR-266447 mmERR_ABORT_TOO_MANY_STATEMENTS_IN_THE_SESSION

Created by Education/Contents Team, last modified on Nov 20, 2014
  • Version
  • Explanation
  • Cause

  • Reference

Version

All versions

Explanation

Unable to execute query. 

Cause

This error occurs if QUERY_STACK_SIZE is insufficient when a statement has not been properly closed.

Action

Check whether the statement is closed properly after query execution or increase QUERY_STACK_SIZE.
