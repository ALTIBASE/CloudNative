FAQ(English) : ERR-61100 rpERR_ABORT_RPC_DUPLICATE_REPLICATION Duplicate replication names. The replication name already exists in the database.

Created by Education/Contents Team, last modified on Oct 20, 2014
  • Version
  • Explanation
  • Cause
  • Action
  • Reference

Version

All versions


Explanation

Unable to create a replication object.

Cause

A replication object using the same object name, IP or PORT number exists.

Action

Neither change the object name nor permit the duplicate IP or PORT number.

Increase the number of network cards as needed.
